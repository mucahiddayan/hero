# Setup

Dieses Projekt benötigt [Docker](https://docs.docker.com/engine/install/) und [Node.js](https://nodejs.org/en/download), um ausgeführt zu werden.

## Installation und Start

1. [Repository](https://github.com/mucahiddayan/hero) auschecken
2. Projekt bauen:
   ```bash
   npm run build
   ```
3. Projekt starten:
   ```bash
   npm run start
   ```
4. Browser öffnen und `localhost:5173` aufrufen

## Hinweise

Es wurden keine externen UI-Bibliotheken verwendet.
